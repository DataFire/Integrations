# @datafire/nba

Client library for NBA Stats

## Installation and Usage
```bash
npm install --save @datafire/nba
```
```js
let nba = require('@datafire/nba').create();

.then(data => {
  console.log(data);
});
```

## Description

The destination for current and historic NBA statistics.
