# @datafire/nytimes_rss

Client library for rss.nytimes.com

## Installation and Usage
```bash
npm install --save @datafire/nytimes_rss
```
```js
let nytimes_rss = require('@datafire/nytimes_rss').create();

nytimes_rss.homePage({}).then(data => {
  console.log(data);
})
```
