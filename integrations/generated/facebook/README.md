# @datafire/facebook

Client library for Facebook

## Installation and Usage
```bash
npm install --save @datafire/facebook
```
```js
let facebook = require('@datafire/facebook').create({
  access_token: "",
  refresh_token: "",
  client_id: "",
  client_secret: "",
  redirect_uri: ""
});

.then(data => {
  console.log(data);
});
```

## Description

Connect to the social network with the Graph API.
