# @datafire/etsy

Client library for Etsy

## Installation and Usage
```bash
npm install --save @datafire/etsy
```
```js
let etsy = require('@datafire/etsy').create();

.then(data => {
  console.log(data);
});
```

## Description

Bring Etsy's handmade marketplace and community into your apps.
